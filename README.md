# 校园树洞 - 微信小程序

一个匿名校园社交平台，让每一次倾诉都能被回应，让每一次互动都被看见。

## 功能概述

### 核心功能
- **发起功能**：用户可匿名发布留言，支持自定义有效期（6h/12h/24h/48h/72h）
- **参与功能**：其他用户可浏览留言并点赞互动
- **结果展示**：点赞排行榜展示最受欢迎的留言，实时票数统计
- **数据上云**：所有数据写入微信云数据库，多人共享实时更新

### 加分功能
- **防重复点赞**：每人只能对同一条留言点赞一次，取消后可重新点赞
- **时间限制**：留言可设置有效期，过期后自动隐藏不再展示
- **分享功能**：支持将留言分享到微信聊天，邀请好友参与互动

## 技术架构

### 页面结构（3个页面）
| 页面 | 路径 | 功能 |
|------|------|------|
| 最新留言 | `pages/index/index` | 留言列表、点赞互动、分享、过期过滤 |
| 发布留言 | `pages/publish/publish` | 匿名发布、有效期选择、匿名代号生成 |
| 点赞排行 | `pages/rank/rank` | 按点赞数排序展示热门留言 |

### 云数据库集合
| 集合名 | 用途 |
|--------|------|
| `treehole_messages` | 存储留言数据（内容、匿名名、点赞数、创建时间、过期时间） |
| `treehole_likes` | 存储点赞记录（userId + messageId，用于防重复点赞） |

### 云函数
| 云函数 | 用途 |
|--------|------|
| `quickstartFunctions` | 提供留言的增删改查、集合创建、openid 获取等云端能力 |

## 快速开始

1. 使用微信开发者工具打开项目
2. 在 `miniprogram/app.js` 中配置你的云开发环境 ID
3. 在云开发控制台创建 `treehole_messages` 和 `treehole_likes` 集合
4. 右键 `cloudfunctions/quickstartFunctions` → 上传并部署
5. 编译运行即可体验
